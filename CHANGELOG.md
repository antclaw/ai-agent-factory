# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-26

### Added
- 🎨 初始版本发布
- ✨ Agent 市场 - 浏览和下载 Agent 模板
- ✨ Agent 工作台 - 配置和运行 Agent
- ✨ Agent 部署 - 多平台发布配置
- 🔧 后端 API - RESTful 接口
- 📚 完整文档 - 开发指南和 API 文档
- 🚀 GitHub Actions - 自动化构建和发布
- 📦 electron-builder 配置 - 多平台打包
- 🎯 4 个内置 Agent - planner, coder, reviewer, researcher
- 🎨 Ant Design UI 组件库集成
- 🌍 国际化支持（中文）

### Documentation
- 📖 README.md - 项目介绍和使用指南
- 📖 DEVELOPMENT.md - 开发环境配置和规范
- 📖 API.md - API 接口文档
- 📖 Agent Skills - Agent 模板文档

### Scripts
- 📜 build.sh - 构建脚本
- 📜 dev.sh - 开发服务器脚本
